Репозиторий состоит из 3-х независимых сервисов: оформление заказа (order-service), обработка платежа (payment-service) и доставка заказа (delivery-service).
В качестве БД используется Postgres, запускаемый локально в docker-контейнере. Описания схем в orders_schema.sql, payments_schema.sql и delivery_schema.sql.
В корне репозитория диаграммы DDD Интернет-магазин_ Декомпозиция по бизнес-возможностям.png и Интернет-магазин_ Декомпозиция по поддоменам.drawio.png.
